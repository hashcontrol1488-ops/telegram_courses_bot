# Telegram Course Bot (aiogram v3)

Умный Telegram-бот для выдачи курсов из текстового файла с:
- категориями и inline-навигацией;
- поиском;
- избранным;
- дневным лимитом;
- баллами/уровнями;
- реферальной системой;
- курсом дня;
- напоминаниями неактивным пользователям.

## Запуск

1. Установите Python 3.11+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте `.env` на основе `.env.example` и укажите `BOT_TOKEN`.
4. Запустите:

```bash
python main.py
```

## Деплой на BotHost

1. Загрузите файлы проекта на BotHost.
2. В переменные окружения добавьте:
   - `BOT_TOKEN` - токен бота;
   - `DB_PATH=bot.db`;
   - `COURSES_FILE=courses.txt`;
   - `REMINDER_CHECK_SECONDS=3600` (по желанию);
   - `ADMIN_USERNAMES=@doubglegwap,@doublegwap` (для команд `/stats` и `/stats_full`);
   - `ADMIN_USER_IDS=123456789` (рекомендуется, самый надежный вариант).
3. Убедитесь, что в корне проекта есть файл `courses.txt` с курсами.
4. Укажите команду установки:

```bash
pip install -r requirements.txt
```

5. Укажите команду запуска:

```bash
python main.py
```

## Команды

- `/start` - приветствие и главное меню
- `/daily` - случайный курс дня (не расходует лимит)
- `/ref` - реферальная ссылка
- `/stats` - короткая статистика (только для админа)
- `/stats_full` - расширенная статистика (только для админа)

## База данных

Автоматически создаются таблицы:
- `users`
- `downloads`
- `favorites`
- `referrals`
- `points`

Файл базы по умолчанию: `bot.db`.

## Формат файла курсов

Поддерживается формат:

```text
Название категории
Скачать Название курса https://link
```

Парсер автоматически собирает структуру:

```python
{
  "категория": [
    ("название", "ссылка")
  ]
}
```
